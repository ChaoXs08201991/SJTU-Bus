SJTU校园巴士时刻表
========

本项目是一个web应用，用于便捷地查看上海交通大学校园巴士的时刻表。

#更新记录
- 2014.9.24: 初级版完成
- 2014.11.2: 优化了移动端的体验
- 2014.11.3: 更正了东川路地铁站的车站位置
- 2014.11.27: 根据后勤处的调整，更新了时刻表的时间
- 2014.12.26: 将打到Github上的所有流量重定向到sina
- 2015.2.10:
  - 添加按钮：“查看最近的车站”
  - 使用FIS来使得项目工程化
